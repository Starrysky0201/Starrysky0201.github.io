数组

数组是由连续无空隙分配的，拥有特定元素类型的对象构成。

这些对象的数目数量（数组大小）在数组生态期间决不改变。

一、数组声明

1. 语法

在数组声明的声明文法中，类型说明符的序列指定元素类型（必须是一个完整对象类型）。而声明符的形式为：

  • [ (static) (限定符) (表达式) ] 属性说明符序列
  • [ (限定符) (static) (表达式) ] 属性说明符序列
  • [ (限定符) * ] 属性说明符序列

表达式 - 任何无逗号运算符的表到时，表明数组中的元素数量

限定符 - const、restrict和volatile限定符的任意混合

属性说明符序列 - (C23)可选的属性列表，应用到被声明的数组

注：未指定大小的VLA的声明符只能出现于函数原型作用域

2. 已知常量大小数组

若数组声明符中的表达式为整数常量表达式，拥有大于零的值，且元素类型是一种拥有已知常量大小的类型（即元素不是VLA）。

3. 非常量长度数组

若表达式不是整数常量表达式，则数组声明器声明一个非常量大小的数组VLA。

每次控制流经过声明时，会求值表达式（而且它必须每次求值为大于零的值），然后分配数组（对应地，VLA的生存期在其声明离开作用域时结束）。

VLA实例的大小不会在其生存期中改变，但在另一次经过同一代码时，它可能被分配不同大小。

VLA必须拥有自动或分配存储期。指向VLA的指针，但不是VLA自身亦可拥有静态存世期。VM类型不能拥有链接。

可修改的类型不是结构体或联合体成员

4. 数组到指针转换

任何数据类型的左值表达式，当用于异于

  • 作为取地址运算符的操作数
  • 作为sizeof的操作数
  • 作为typeof和typeof_unqual的操作数（C23起）
  • 作为用于数组初始化的字符串字面量

特定的语境时，会经历到指向其首元素指针的隐式转换。结果为非左值。

当数组类型用于函数参数列表时，它会转换成对应的指针类型：int f(int a[2])和int f(int* a)声明同一个函数。因为函数实际参数类型为指针类型，使用数组实参的函数调用会进行一个数组到指针转换；被调用函数无法获得实参数组的大小，必须显式传递：

二、数组定义

初始化数组类型的对象时，初始化器必须是一个字符串字面量，或是一个花括号环绕的被初始化数组成员列表：

  • = 字符串字面量
  • = {表达式, … }
  • = {}

1. 字符串初始化

字符串字面量的连续字节，或宽字符串字面量的连续字符，包含空终止字节/字符，会初始化数组的元素。

2. 花括号环绕列表初始化

当数组以花括号环绕的初始化器列表初始化时，首个初始化器初始化序号为零的数组元素，而每个后继的初始化器所初始化的数组元素，序号比前一个初始化器所初始化的多一。

指派符导致接下来的初始化器初始化指派符所描述的数组元素。然后初始化以向前顺序持续，从指派符所描述元素的下一个元素开始。
